Випускна робота на тему: "Популяризація роману "Улісс" Джеймса Джойса".
Виконав: учень 11-В класу Дрововозов Сергій.
Платформа розробки: HTML, CSS, Node (версія: 22.11.0), Angular CLI (версія: 18.2.11).
Апаратні вимоги до ПК: двоядерний процесор, 4 Гб оперативної пам'яті, внутрішня пам'ять більше 8 ГБ.
Для запуску проєкту треба мати встановлений Node та Angular CLI.
Скачати найновшу версію Node можна за посиланням: https://nodejs.org/uk.
Перевірити, яка версія Node була встановлена можна у командному рядку за допомогою команди: node -v.
Встановити Angular CLI можна у командному рядку за допомогою команди: npm install -g @angular/cli.
Перевірити, яка версія була Angular CLI встановлена можна у командному рядку за допомогою команди: ng version.
